# myAmpApp
This is infinite scroll nodejs app with AMP 

## Technologies
* Node.js
* Express
* AMP
* Pug Templating

### Installation

Clone the repository,
then delete the "node_modules" folder

Install the dependencies

```sh
$ npm install
```
Run app

```sh
$ npm start
```
Listening on port 3000
(http://localhost:3000)
